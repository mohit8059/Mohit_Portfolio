NIFTY Stock Market Dashboard

Power BI dashboard created to analyze NIFTY stock performance, price trends, trading volume, returns, top gainers and top losers.

Project Overview

This dashboard helps users track stock market movement, compare companies, identify top gainers/losers and analyze price trends.

Tools Used
Power BI Excel DAX Power Query Data Visualization
KPIs
  • Total Stocks
  • Total Volume
  • Average Close Price
  • Highest Close Price
  • Return %

Dashboard Features

  • Market performance overview

  • Analyzed stocks with highest trading volume.
