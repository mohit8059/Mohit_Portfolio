IPL Analytics Dashboard

Power BI dashboard created to analyze IPL team performance, player statistics, runs, wickets, strike rate, top players and match trends.

Project Overview

This dashboard helps users analyze IPL performance across teams, players, seasons and match results using interactive visuals.

Tools Used
Power BI Excel DAX Power Query Data Visualization
KPIs
  • Total Runs
  • Total Wickets
  • Average Strike Rate
  • Total Matches
  • Top Players
